# LookAhead

LookAhead es una aplicación que combina realidad aumentada, visión por computadora e inteligencia artificial para simular cambios estéticos (como maquillaje o cortes de cabello) antes de realizarlos. Su objetivo es ayudar a los usuarios a visualizar resultados en tiempo real, reduciendo la incertidumbre y el desperdicio de recursos.

## Descripción del proyecto

La apariencia personal influye significativamente en la percepción social y la autoconfianza. LookAhead busca mejorar la experiencia de decisión estética mediante simulaciones visuales realistas y personalizables.

El sistema utiliza MediaPipe para la generación de la malla facial y OpenCV para el procesamiento de imágenes, permitiendo aplicar filtros estéticos sobre el rostro de manera fluida y en tiempo real.

## Características principales
- Aplicación de filtros de maquillaje y estilo en tiempo real.
- Visualización dinámica con seguimiento facial.
- Personalización de intensidad y tipo de filtro.
- Compatible con Windows (versión actual).
- Desarrollado en Python con MediaPipe y OpenCV. 
- Interfaz intuitiva y adaptable.

## Objetivos
- Ofrecer visualizaciones estéticas realistas y precisas.
- Permitir experimentación sin riesgos físicos ni económicos.
- Desarrollar versiones multiplataforma (Windows y Android).
- Mantener un desempeño estable incluso en dispositivos de bajos recursos.

## Estado del proyecto
Actualmente, LookAhead se encuentra en una fase temprana de desarrollo.
La versión de escritorio funciona correctamente, pero aún se trabaja en la calidad visual de los filtros y en la futura versión móvil para Android.
